-----------

2. KABISAT
KABISAT 'ALGORITMA'
=========================
SIMPAN nilai 'Tahun'
SIMPAN nilai 'Tahun Kabisat'=> tidak disi dulu nilainya ,kita akan menghitungnya terlebih dahulu

MENGHITUNG nilai'Tahun kabisat' yaitu nilai 'Tahun' MOD 4
TAMPILKAN  hasil nilai'Tahun Kabisat'

JIKA hasil dari nilai 'Tahun Kabisat' sama dengan 0 maka
9.a.Lanjut ke LANGKAH BERIKUTNYA
9.b Menampilkan (BUKAN TAHUN KABISAT)

SIMPAN Hasil nilai dari 'Tahun kabisat' sebelumnya
HITUNG nilai 'Tahun kabisat' yaitu Hasil nilai'Tahun Kabisat' sebelumnya MOD 100

JIKA hasil dari nilai 'Tahun Kabisat' sama dengan 0
17.a Menampilkan (TAHUN KABISAT)
17.b Lanjut ke LANGKAH BERIKUTNYA

SIMPAN Hasil nilai dari 'Tahun kabisat' sebelumnya
HITUNG nilai 'Tahun kabisat' yaitu Hasil nilai'Tahun Kabisat' sebelumnya MOD 400

JIKA hasil dari nilai 'Tahun Kabisat' sama dengan 0 MAKA
25.a Menampilkan (TAHUN KABISAT)
25.b Menampilkan (BUKAN TAHUN KABISAT)

Selesai

KABISAT'pseudocode'
==========================
STORE var 'Tahun'
STORE var 'Tahun Kabisat' without any value

CALCULATE 'Tahun Kabisat' = var 'Tahun' MOD 4
DISPLAY result 'Tahun Kabisat'

IF result 'Tahun Kabisat' = 0 THEN
DISPLAY (TAHUN KABISAT)
ELSE
(DO NEXT STEP)
END IF

STORE result var 'Tahun Kabisat' before
CALCULATE var 'Tahun Kabisat' = var result 'Tahun Kabisat' before MOD 100
DISPLAY result 'Tahun Kabisat'

IF result 'Tahun Kabisat' = 0 THEN
DO NEXT STEP
ELSE

SISWA 'ALGORITMA'
==========================
SIMPAN 'jumlah murid' adalah 40
SIMPAN 'murid' adalah salah satu bagian dari jumlah murid
SIMPAN 'bu guru'adalah orang yang memeriksa
SIMPAN 'hasil riset'adalah nilai dari memeriksa kuku (BENAR ATAU SALAH)

KETIKA 'murid' diperiksa oleh 'bu guru' dari 'murid' ke 1
hingga 'murid' ke 40
SIMPAN nilai 'hasil riset' dari 'murid' yang diperiksa

JIKA 'murid' memiliki kuku panjang adalah BENAR MAKA
TAMPILKAN (GURU AKAN MENGHUKUM MURIDNYA)
NAMUN
(GURU SANGAT MEMUJI MURIDNYA)

SELESAI

SISWA 'pseudocode'
========================
STORE sum 'murid' is 40
STORE 'murid' is unit from 40
STORE 'bu guru'
STORE 'hasil riset' is var (TRUE OR FALSE)

WHILE 'murid' DO 'bu guru' FROM 1 to 40
STORE 'hasil riset'

IF 'murid' memiliki kuku panjang is TRUE THEN
DISPLAY (GURU AKAN MENGHUKUMNYA)
Else
(GURU AKAN SANGAT MEMUJINYA)

END IF
